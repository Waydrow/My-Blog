使用日志API

Java :

  • 错误日志-> System.err.println()
  • 普通日志-> System.out.println()

Android :

  • 错误信息-> Log.e()
  • 警告信息-> Log.w()
  • 普通信息-> Log.i()
  • 调试信息-> Log.d()
  • 无用信息-> Log.v()
